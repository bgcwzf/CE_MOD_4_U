Hi,这个文件夹中，用于存放为《戴森球计划》来编译MOD所需的各种.NET程序集文件(DLL)。

在编译各个项目之前，需要将当前版本的《戴森球计划》所使用的各种程序集文件，复制到这里。

主要有以下步骤：
1.打开“DSPGAME.exe”所在的文件夹
2.打开“DSPGAME_Data\Managed”文件夹
3.把此“Managed”文件夹中所有的DLL文件，复制到当前README.md文件所在的这个文件夹中。

后续如果需要自行编写MOD，则直接在新的项目中引用在本文件夹中的DLL即可。

*请注意：在MOD项目工程文件中，添加的程序集“引用”项，最好是将它的“复制本地”配置项的值，设置为FALSE
